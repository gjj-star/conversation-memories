## 课余技术补课（按优先级）

### 🔴 第一优先级：Linux 复活 + 强化（立刻开始，2-3 周）
- 装 WSL2 或 VirtualBox + Ubuntu
- 日常操作恢复：`ssh` `scp` `grep` `find` `chmod` `systemctl` `journalctl`
- 网络诊断练手：`curl -v` `ping` `traceroute` `nslookup` `netstat` `ss`
- Shell 脚本实战：自动备份脚本、日志清理脚本、健康检查脚本
- **目标**：能在客户服务器上独立排查"为什么服务起不来"

### 🟠 第二优先级：Docker（3-4 周）
- 理解镜像、容器、卷、网络、Dockerfile
- 把一个 Python 脚本打包成 Docker 镜像
- docker-compose 编排"应用 + 数据库 + Nginx"三件套
- **目标**：能用 Docker 把任意应用跑起来

### 🟡 第三优先级：后端基础（4-6 周）
- FastAPI（Python 生态，入门快）写 REST API
- SQLAlchemy + PostgreSQL CRUD 操作
- `requests`/`httpx` 调第三方 API，处理认证、分页、错误重试
- 写一个"从客户 API 拉数据 → 清洗 → 写入数据库"的集成脚本
- **目标**：能独立把 A 系统的数据搬到 B 系统

### 🟢 第四优先级：网络 + 安全（碎片时间持续学）
- TCP/IP 基础、DNS 原理
- HTTP 深入：请求/响应头、状态码、缓存、CORS
- TLS/SSL 证书链、自签名 vs CA 签发
- OAuth 2.0 授权码流程（业财系统必碰）
- SSH 隧道/端口转发（客户内网必需）
