乔治·索罗斯在《金融炼金术》（1987）中提出的核心理论：在社会事务中，参与者的认知改变事态，事态反过来修正认知——双向反馈环路。

## 两个核心函数
- **认知函数（Cognitive Function）**：参与者试图理解局势 → 形成看法
- **参与函数（Manipulative Function）**：基于看法行动 → 改变现实
- 两个函数同时运行、互相喂料：`看法 → 行动 → 改变现实 → 修正看法 → ...`

## 与传统经济学的对立

| 传统经济学 | 反身性 |
|---|---|
| 价格反映基本面 | 价格改变基本面 |
| 参与者理性 | 参与者带有不可避免的认知偏见 |
| 市场趋向均衡 | 市场趋向极端，均衡是特例 |
| 预测独立于被预测对象 | 预测本身就是系统的一部分 |

## 繁荣-崩溃八步循环
1. 资产"看起来很值钱" → 银行大量放贷 → 价格上升
2. "果然是好资产" → 更多购买 → 价格继续涨
3. "只会涨不会跌" → 杠杆加码 → 价格偏离基本面
4. 小事件触发怀疑 → 聪明人撤 → 价格微跌
5. "跌了！跑！" → 恐慌抛售 → 暴跌
6. "一文不值" → 信贷冻结 → 暴跌远超基本面
7. 危机蔓延 → 政府救市 → 新规则形成
8. "学到了教训" → 建立新制度 → 直到下一轮

## 经典例子
- **银行挤兑**：谣言本来不真 → 储户取钱 → 银行真倒闭了（自我实现）
- **股评预测**：分析师说涨 → 大家买入 → 真涨了（不是预测准，是参与制造）
- **技术指标**：所有人用同一指标 → 触发时同时买卖 → 指标制造了价格走势
- **民调**：某候选人领先 → 支持者觉得稳了不出门投票 → 反而输了（自我否定）

## 与六爻的关系
卦进入参与者认知 → 认知驱动行为 → 行为创造结果 → 很难分清是卦"看"到了结果还是卦"制造"了结果。所有对"有思考能力的参与者"的预测都陷在这个困境里。

## 外延
政治、教育（皮格马利翁效应）、人际关系、AI 模型自噬——任何"被预测对象因预测而改变"的领域。
