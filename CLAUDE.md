# CC Learning — 对话自动记录仓库

## 自动记录规则

**每轮对话必须执行以下流程：**

1. **记录**：将用户的问题和对话摘要追加到 `conversation-log.md`，格式为：
   ```
   ## [日期 时间] 问题摘要
   - **用户问**：<用户的问题>
   - **回答摘要**：<简要总结你的回答>
   ```

2. **提交**：将记录 git commit，commit message 格式：`log: <问题摘要>`

3. **推送**：`git push` 到 GitHub remote

4. **告知**：在你的回复开头（或结尾）告知用户：**📝 已记录并推送至 GitHub。**（如果 push 失败则说明原因）

## 注意事项
- 这个仓库只存放对话记录，不存放项目代码
- `.omc/` 目录已加入 .gitignore
- 确保 GitHub remote 已配置（详见全局配置）
