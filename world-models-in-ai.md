## 世界模型 (World Model)

本质是一个**状态转移预测器**——给定当前状态 + 动作，预测未来的状态。不靠物理引擎算力学方程，而是从真实数据中学会物理规律。

核心能力：动作后果预演、失败轨迹生成、价值评估。

## 世界模型 vs 大语言模型

| 维度 | LLM | 世界模型 |
|------|-----|---------|
| 学什么 | 语言统计规律 | 环境状态转移规律 |
| 预测什么 | 下一个 token | 下一个状态（画面/物理量） |
| 验证方式 | 文本一致性（无物理真值） | 物理世界真实反馈（可证伪） |
| 数据成本 | 互联网文本，近乎免费 | 真机 rollout + 多视角视频，极贵 |
| 成熟度 | 大规模应用 | 早期（类似 2020 年的 LLM） |

**相同点：** 都是预测式自监督学习；都有 Scaling Law 涌现；都是"压缩器"；都可当"心智模拟器"。

**关键区别：** LLM 活在"共识现实"（文本分布），世界模型活在"物理现实"（可被物理世界证伪）。

**终极融合方向：** LLM + 世界模型 = 既能思考又能行动的完整智能体（LLM 给 AI "想"的能力，世界模型给 AI "感知物理世界并预判后果"的能力）。

## 后训练 (Post-Training)

预训练完成后让模型"变得好用"的所有训练阶段，按顺序：

1. **SFT（监督微调）** — 看标准示范，学会"照着做"。只学正确答案，不知好坏。
2. **RLHF / RLAIF** — 人类对多个输出打分排序，模型学会偏好区分。
3. **在线 RL** — 模型自己尝试，环境给 reward 反馈，自主探索进化。

类比：预训练=读教科书 → SFT=看老师示范 → RLHF=老师点评打分 → 在线RL=自己做实验改进

**具身智能的后训练特别难：** 试错成本高（硬件损耗）、反馈慢（物理动作耗时）、信号模糊（无编译器判定对错）、数据难扩（不能爬取）。

## VLA 与世界模型结合（DW0.5 案例）

**VLA**（Vision-Language-Action）：具身智能中的视觉-语言-动作模型，问题在于物理理解缺失、泛化弱、长时序规划不足。

**DW0.5**（原力灵机，2025.07）：具身世界模型，接入 DFOL 2.0 后训练框架。
- 三大模块：Video Expert（预演未来）+ Action Expert（动作强先验）+ Value Expert（打分反馈）
- 工作流：VLA 生成动作 → DW0.5 虚拟预演成功/失败 → Value Expert 打分 → RL 更新策略
- 效果：真机数据需求降 60%，训练成本降 40%；任务成功率大幅提升（打气球 10%→90%）
- 在 EWMBench、WorldArena 基准测试获全球 SOTA

**核心洞察：** 世界模型充当"虚拟教练场"，把 RL 的试错循环从物理世界转移到"模型的想象中"。类似棋手在脑中推演棋局，不需要每步都摆真棋盘。
